# Skills aggregator loop

`operations/ledger.json` is the one current-work authority for discovery,
review, normalization, catalog, risk, packaging, publication preparation, and
critique.

```text
inspect source/catalog → select one ready item → act in one role → verify →
record evidence and transition → stop
```

Review, provenance, license, risk, and catalog integrity outrank growth.
Reference first, summarize by default, copy rarely, normalize only when useful,
package only when safe, and publish only with clear provenance and authority.
Empty ready state is a clean stop, not a reporter task.
